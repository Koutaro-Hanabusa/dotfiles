2. 質問と回答のやり取りを.mdファイルとして、~/.nb/home/knowledge が存在するのでそこの配下に記録・蓄積します。
仕事用のPCでは内容を~/.nb/work/knowledge に記述して欲しいです。仕事用のPCには、~/.is_work_pcがあります。

```bash
# 通常のPC（home notebook）の場合
nb add home:knowledge/ -c "内容"

# 仕事用PC（work notebook）の場合
nb add work:knowledge/ -c "内容"

# 既存のメモを編集する場合
nb edit home:knowledge/<ファイル名>
# または
nb edit work:knowledge/<ファイル名>
```
