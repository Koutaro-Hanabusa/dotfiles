# Job Hunter — ファンテック転職支援スキル

## ユーザープロフィール

- **職種**: フロントエンドエンジニア
- **興味業界**: ファンテック（Fan Tech） — ファンクラブ・ファンエンゲージメントPF構築
- **志向**: BtoC PF型、エンタメ×テック、ファンが直接触れるUI/体験の最前線
- **注目企業**: Gaudiy、SKIYAKI
- **技術関心**: React / Next.js / TypeScript 周辺のモダンFEスタック

## 機能

### 1. 企業リサーチ (`research`)

指定された条件で企業を検索・分析する。

**収集項目**:
- 企業名（日英）
- 主要プロダクト・サービス
- 技術スタック（特にフロントエンド）
- クライアント・導入実績
- 資金調達・上場状況
- 採用ページURL
- フロントエンドエンジニアの求人有無
- 年収帯（公開されている場合）

**検索キーワード例**:
- 「ファンテック 企業」「fan engagement platform Japan」
- 「ファンクラブ プラットフォーム エンジニア採用」
- 「エンタメDX フロントエンド 求人」

**出力**: 企業情報を表形式でまとめ、Gaudiy/SKIYAKIとの類似度を評価する。

### 2. 求人検索 (`search`)

特定企業または条件で求人を検索する。

**検索先**:
- 企業の採用ページ（直接URL）
- Green, Wantedly, Findy, HERP 等の求人媒体
- 「{企業名} フロントエンドエンジニア 求人」でWeb検索

**収集項目**:
- ポジション名
- 必須スキル / 歓迎スキル
- 年収帯
- 勤務形態（リモート可否）
- 応募URL

### 3. 応募管理 (`track`)

応募状況をトラッキングファイルで管理する。

**ファイル**: `~/.claude/projects/-Users-1126buri/memory/job_tracking.md`

**管理項目**:

| 項目 | 内容 |
|------|------|
| 企業名 | - |
| ポジション | - |
| ステータス | 興味あり / 応募準備中 / 応募済 / 書類通過 / 面接中 / 内定 / 見送り |
| 応募日 | - |
| 次のアクション | - |
| メモ | - |

### 4. 面接対策 (`prep`)

指定企業の面接対策を行う。

- 企業の技術ブログ・テックスタックを調査
- 想定質問リスト作成（技術質問 + カルチャーフィット）
- 「なぜファンテック？」「なぜ御社？」のストーリー整理
- ポートフォリオで強調すべきポイントの提案

### 5. 市場分析 (`market`)

ファンテック業界の動向・トレンドを調査する。

- 業界の最新ニュース・資金調達情報
- 技術トレンド（Web3、AI、リアルタイム通信 etc.）
- 給与相場

## 使い方

```
/job-hunter                    → メニュー表示
/job-hunter research {条件}     → 企業リサーチ
/job-hunter search {企業名}     → 求人検索
/job-hunter track              → 応募状況一覧
/job-hunter prep {企業名}       → 面接対策
/job-hunter market             → 市場分析
```

## 実行ルール

1. 返答は必ず**日本語**で行う
2. Web検索が可能な場合は最新情報を取得する
3. 企業情報は必ず**公式サイト・採用ページ**のURLを添える
4. 応募管理の更新時はメモリファイルも更新する
5. 面接対策では企業の技術ブログがあれば必ず確認する
6. リサーチ結果は表形式で見やすくまとめる
